# Breathing Bubble

Minimal breathing exercise with a single animated focal point.

## Scripts

- `npm install` installs project dependencies.
- `npm run dev` starts the local Vite dev server.
- `npm run build` creates a production build in `build/`.
- `npm run preview` serves the production build locally.

## Notes

- The app is intentionally minimal and centered on one calming interaction.
- Production output is written to `build/` so it can continue to be published as a static subproject.
